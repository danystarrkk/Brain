------------

## Ghidra - Reverse Engineering Tool

|Categoría|Comando/Función|Descripción|Atajo Teclado|
|---|---|---|---|
|**Proyecto**|`File > New Project`|Crear nuevo proyecto|`Ctrl + N`|
||`File > Import`|Importar archivo binario|`Ctrl + I`|
||`File > Save Project`|Guardar proyecto|`Ctrl + S`|
|**Análisis**|`Analysis > Auto Analyze`|Análisis automático||
||`Analysis > One Shot`|Análisis específico||
||`Window > Function Graph`|Grafo de funciones||
|**Navegación**|`Window > Code Browser`|Navegador principal|`Ctrl + Shift + C`|
||`Window > Symbol Tree`|Árbol de símbolos|`Ctrl + Shift + S`|
||`Window > Listing`|Vista de desensamblado|`Ctrl + Shift + L`|
|**Edición**|`Edit > Undo/Redo`|Deshacer/Rehacer|`Ctrl + Z/Y`|
||`Edit > Rename`|Renombrar variable/función|`L`|
||`Edit > Comment`|Agregar comentario|`;`|
|**Vistas**|`Window > Decompile`|Vista de decompilación|`F5`|
||`Window > Memory Map`|Mapa de memoria||
||`Window > Data Type Manager`|Gestor de tipos de datos||
|**Scripting**|`Window > Script Manager`|Gestor de scripts||
||`Analysis > Run Script`|Ejecutar script||
||`Python > Interactive`|Consola Python||
|**Búsqueda**|`Search > For Strings`|Buscar strings|`Ctrl + F`|
||`Search > For Functions`|Buscar funciones|`Ctrl + Shift + F`|
||`Search > For Patterns`|Buscar patrones||
|**Funciones**|`Function > Create Function`|Crear función|`F`|
||`Function > Edit Function`|Editar función|`Ctrl + F`|
||`Function > Stack`|Editor de stack||
