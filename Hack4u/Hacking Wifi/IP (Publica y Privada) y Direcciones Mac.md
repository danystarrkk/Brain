Nosotros tenemos dos tipos de tipos de ip, una publica y una privada.

## IP Privada

Las ip privadas son las designadas de manera local por nuestro router, este tipo de ip solo las podemos usar cuando nos encontramos dentro de una red privada. Este tipo de ip no puede ser usada por un atacante que no se encuentre en nuestro mismo segmento de red.


## IP Publica

Las ip publicas son únicas y se asignan a un dispositivo el cual tenga acceso a internet y suele ser invisible de manera globlal, este tipo de ip también nos sirve al igual que la privada para una comunicación pero mediante internet, Las direcciones IP públicas son asignadas por organizaciones regionales de Internet, conocidas como Registros Regionales de Internet (RIRs).


## Dirección Mac

Las direcciones Mac no son mas que identificadores únicos que se les asigna a los dispositivos. Esto es cierto pero solo hasta cierto punto ya que nosotros podemos cambiar o falsificar la Mac de un dispositivo.
