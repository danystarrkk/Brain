## Qué es internet?

**Internet** -> Conjunto de computadoras interconectadas que permiten el intercambio de información mediante un conjunto de normas llamadas protocolos, La primera red se llama **ARPANET**, este con el tiempo crece y cerca de los 90 crece y se rige bajo la Word Wide Web en  por Time Berners-Lee, se consolido un estándar llamado W3C que seria un consorcio.

De todo esto nacen tres componentes fundamentales:

- HTML -> (Hyper Text Markup Language) -> el lenguaje para crear páginas web.
- HTTP  -> (Hyper Text Transfer Protocol) -> protocolo para transferir información entre computadores.
- URL -> (Uniform Resource Locator) -> La dirección única que permite acceder a cada página.

## Qué es HTML?

HTML es el lenguaje utilizado para estructurar el contenido de una página web. No es un lenguaje de programación, sino un lenguaje de marcado que usa etiquetas. Nos permite definir la estructura básica de un sitio web.

### Evolución de HTML

Desde su creación paso por varias versiones:

- HTML 2.0
- HTML 3.2
- HTML 4.0.1
- XHTML
- HTML5

## Qué es HTTP?

Es un protocolo de comunicación que permite la transferencia de información entre un cliente (navegador) y un servidor web. Entonces el cliente genera una solicitud (request) y el servidor genera la respuesta (response).

El ciclo es -> solicitud -> procesamiento -> respuesta.

El protocolo HTTPS es una versión segura el cual agregar la capa de cifrado mediante SSL/TLS, l oque significa que la información viaja protegida y no puede ser leída por terceros.
