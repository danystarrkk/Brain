# Que es la UI?

Una interfaz de usuario en ingles **User Interface**, La interfaz de usuario es importante para una buena experiencia de usuario. cuentan con los elementos de control de una interfaz , que son posibles para el usuario mediante las interacciones y la facilidad de uso a partir de la experiencia de dicho usuario.

## Tipos de UI

1. **Command Lines Interfaces (CLI)**: La linea de comandos es una de las interfaces mas primitivas y la mas usada, ademas de que no va a desaparecer ya que es la base de los Tipos de UI.
2. **Text User Interface (TUI)**: Las TUI marcan el paso de meras interfaces de lineas de comando a interfaces gráficas de usuarios, estas interfaces utilizan las pantalla s como un medio y no como una pantalla de lineas planas.
3. **Graphical User Interface (GUI)**: Esto es el estándar actual mas usado en la actualidad, tiene una combinación de colores y diseños intuitivos para permitir a los usuario mediante colores o indicadores guiar al usuario que hace cada cosa.
4. **Voice user Interface (VUI)**: Estas interfaces también permiten interactuar con los ordenadores mediante control de voz, muchos ejemplos serian Siri, Alexa, Bixby, Google y mas. Este en muchas ocasiones nos permite una mejor manipulación por parte del usuario.
5. **Natural User interface (NUI)**: La comunicaciones con el ordenador se establece en un modo especial intuitivo a través de un Natural User Interface. Esta es una evolución de la interfaz gráfica de usuario y de la Voice user interface, por lo tanto la interfaz reacciona tanto con gestos como movimientos y al lenguaje.
6. **Otros**: Tenemos mas dispositivos como pueden ser:
	- **Tangible User Interface (TUI)**: son interfaces de usuario figurativas y personalizables. es decir la interacción con el se lleva a cabo a través de objetos físicos.
	- **Perceptual User Interfaces (PUI)**: Todavía están vía de desarrollo y se ven impulsadas por la ciencia, de ahí que se trate de interfaces de usuario controladas por percepción.
	- **Brain Computer interfaces (BCI)**: ya no forman parte de la ciencia ficción y es que con ayuda de electrodos se puede medir ondas cerebrales y hacer que los algoritmos las traduzcan en comandos de control.
