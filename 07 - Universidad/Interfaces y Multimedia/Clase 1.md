# Clase 1

## Interfaces Gráficas de Usuario

**Interfaz** -> Es el punto de contacto físico o digital entre el usuario y la máquina:

- **Años 70** -> Todo era CLI (Interfaz de Linea de comandos)
- **1973 (Xerox Alto)** -> Nace la primero GUI con metáforas de escritorio (carpetas y ventanas).
- **1984 (Apple Macintosh)** -> La interfaz se vuelve masiva y amigable popularizando el mouse.
- **Hoy** -> Pasamos de lo visual a lo táctil y ahora a interfaces de voz (VUI) o gestuales.
