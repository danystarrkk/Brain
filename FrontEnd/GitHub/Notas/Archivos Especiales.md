# Archivos Especiales
## .gitkeep
El archivo .gitkeep no es mas que un convenio ya que de manera oficial no lo encontraremos en la documentación de git. Por lo tanto nosotros a este archivo le podemos dar cualquier nombre pero repitiendo este nombre es de un convenio y usado para que git agregue esas carpetas sin información que tenemos. Algunos proyectos requieren capetas vacías por el momento que luego se llenaran de información pero git no almacena capetas vacías y para que lo haga usamos el archivo.

## .gitignore
El archivo .gitignore es un archivo que le indica a git que archivos y directorios no queremos darle seguimiento.
