# Git
Git es un sistema de control de versiones, pero que es un sistema de control de versiones?. Un sistema de control de versiones nos permite crear una especie de maquina del tiempo, donde tenemos diferentes tiempos por los cuales nosotros podemos movernos, una forma gráfica de verlo es:

![[Pasted image 20250125071209.png]]

Nosotros podemos movernos entre todos esos tiempos.

En Git no existe el termino proyecto, nosotros en Git tenemos repositorios. Git trabaja con algo que se llama repositorios distribuidos, esto quiere decir que en un equipo de trabajo cada integrante tiene una copia del repositorio la cual puede modificar y le permite evidenciar sus cambios sin tener que enviarlos al repositorio centra, esto nos permite también una forma de mantener diferentes copias de seguridad de los repositorios.
De forma general git nos permite realizar copias (commits) de la información que lo requiera, esto se almacenara y nos permitir movernos entre diferentes copias con seguridad.
