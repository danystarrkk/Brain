# Recomendaciones en Código

En JavaScript no es necesario el uso del `;` en nuestro código, esto lo podemos poner como no y JavaScript no nos reportara ningún error.

- La recomendación es usar el `;` al final de una línea para poder tener un código ordenado.
- Usar solo una Instrucción por línea.
- Dentro de las funciones es recomendable el uso de la tabulación para que este se puede alinear de forma correcta.
