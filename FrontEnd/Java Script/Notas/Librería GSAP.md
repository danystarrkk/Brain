# GSAP
GSAP es una biblioteca de JavaScript para crear animaciones de alto rendimiento que funcionan en **todos** los navegadores principales. Anima CSS, SVG, canvas, React, Vue, WebGL, colores, cadenas, rutas de movimiento, objetos genéricos... ¡todo lo que JavaScript pueda tocar! Ninguna otra biblioteca ofrece una secuenciación tan avanzada, confiabilidad y control tan estricto a la vez que resuelve problemas del mundo real en millones de sitios. GSAP soluciona innumerables inconsistencias del navegador; tus animaciones **simplemente funcionan**. En esencia, GSAP es un manipulador de propiedades de alta velocidad que actualiza valores a lo largo del tiempo con extrema precisión. ¡Es hasta 20 veces más rápido que jQuery! Consulta https://greensock.com/why-gsap/ para saber qué hace que GSAP sea tan especial.
