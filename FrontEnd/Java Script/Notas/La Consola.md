# La consola en JavaScript

Tenemos que saber que todos los navegadores tienen una consola, puede estar en diferentes lugares, pero en la gran mayoría de veces la vamos a encontrar en Inspeccionar del navegador.

La consola es otra de esas herramientas fundamentales, especialmente cuando vamos iniciando en JavaScript por lo que es muy importante aprender a usarlas, por lo tanto, en la consola podemos:

- Crear Variables.
- Crear un Arreglo.
- jugar con diferentes funciones.
- Generar o Crear errores.
- Generar o Crear advertencias.

También podemos enviar datos a la consola de la siguiente forma:

```jsx
console.time('Tiempo');

console.warn("Eso no se puede realizar");
console.warn("Eso no se puede realizar");
console.warn("Eso no se puede realizar");
console.warn("Eso no se puede realizar");
console.warn("Eso no se puede realizar");
console.warn("Eso no se puede realizar");
console.warn("Eso no se puede realizar");
console.warn("Eso no se puede realizar");
console.warn("Eso no se puede realizar");

console.timeEnd('Tiempo');
```
