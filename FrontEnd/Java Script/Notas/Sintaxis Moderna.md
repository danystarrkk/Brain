# Sintaxis Moderna
## Variables con `let`
```jsx
let producto = "Tablet";

// Reasignar el valor

producto = "Monitor";
producto = 20;

console.log(producto);

// Creando variables sin valor
let precio;

precio = 40;
console.log(precio);

```

## Variables con `const`
```jsx
// No podemos reasignar valores cuando usamos const.
const producto = 20;

// Las variables siempre inician con un valor.

const valor = 10;
```
