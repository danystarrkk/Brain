## Sombras en Css
Nosotros podemos aplicar sombras a diferentes objetos como imágenes o  bloques, para esto vamos a usar la siguiente propiedad:
```css
.button{
	box-shadow: 0.2rem 0.4rem 1rem rgba(0,0,0,0.1);
	/*<valor en x> <valor en y> <rango en el que se difumina> <color>*/
}
```

Bueno para las sombras y colores de degradados podemos usar aplicaciones como [cssmatic](https://www.cssmatic.com/es/box-shadow) y tan solo es copiar el código que este nos da y pegarlo en donde lo queremos.
