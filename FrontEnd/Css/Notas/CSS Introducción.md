# CSS Introducción
CSS es el encargado de los colores, tamaños, espacios o animaciones, por lo que mientras html se encarga de la información css se encargara de dar estilo a toda esa información. css Son hojas de estilo en cascada que nos permite dar a nuestro código html dar estilo por lo que se encargara de:

- Tamaños y tipos de fuentes
- Colores
- Espacios
- Márgenes
- Adaptación de dispositivos
- Animaciones

A css se lo conoce como hojas de estilo en cascada porque siempre aplicara el ultimo valor que encuentre sobre un mismo selector.
Tenemos que pensar que en realidad css es fácil, no es nada del otro mundo y un ejemplo de su sintaxis es:

```css
p {
 color: blue;
}

```

bueno como vemos no es nada complicado aunque si se ve extraño por lo que para comprender esto mejor veamos la anatomía de css.
