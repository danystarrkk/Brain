# Background position

Esto será esencial para poder ubicar de forma correcta Background en nuestra web, podemos pasarle tanto uno como dos valores, esto con el objetivo de ubicar diferentes cosas de fondo, lo que se le pasa a este método son 1, 2 o 4 valores de la siguiente forma:

```css
background-position: top; /*cuando es 1 valor*/
background-position: top 50px; /*cuando son dos valores*/
background-position: bottom 50px right 100px; /*cuando son 4 valores*/
```

Algo a tener muy presente es que le pasamos el punto desde el cual moverá el porcentaje de px asía el lado contrario. el valor que se mueve puede ser cualquiera incluyendo los vh o vw.
