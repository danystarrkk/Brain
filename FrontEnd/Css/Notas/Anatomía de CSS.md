# Anatomía de CSS

```css
p {

}

```

bueno este primer parámetro no es mas que la selección de la etiqueta de html que se esta usando y en este caso usamos una p ya que es la etiqueta de párrafo en html.

```css
p {
	color: blue;
}

```

Todo lo que tenemos dentro se llaman sets de valores o que se conocen como llave y valor, a la llave se lo conoce como propiedad en este caso es el color y tenemos el valor que es blue, tenemos que saber que el en todo lo que escribamos que no sea la selección de etiqueta tiene que terminar en un punto y coma.
