# Normalice

De manera general con esto logramos que nuestra web se vea casi igual en todos los navegadores, para esto vamos a irnos a la pagina de [normalice](https://necolas.github.io/normalize.css/) y veremos lo siguiente:

![[Pasted image 20250124162622.png]]

Damos en download y veremos:

![[Pasted image 20250124162653.png]]

Una recomendación es siempre en esta pagina desactivar el traductor si lo tenemos habilitado para no generar problemas. Copiamos todo lo que tenemos en esa pagina y lo pegamos en un archivo con el nombre de normalice.css en nuestra carpeta css.
ya con esto lo que falta por hacer es agregarlo a nuestro html de la siguiente manera:

![[Pasted image 20250124162743.png]]

Una recomendación es que normalice sea la primera librería que cargue para evitar problema y con esto tenemos ya listo todo.
