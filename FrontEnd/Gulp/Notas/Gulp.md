# Gulp

Gulp nos permite automatizar tareas que son repetitivas en el desarrollo web, como lo son Compilar JavaScript y SASS, crear imágenes más ligeras o la versión webp minificar y mejora el código para producción.

Bueno, para Usar Gulp es necesario de Node.js y NPM.

Gulp utiliza JavaScript por lo que es bastante sencillo usarlo, también vamos a tener lo que se llama Pipes, esto es una acción que se ejecuta una vez finaliza una ejecución comienza otra, el orden en que se ejecuta es definido por ese pipe.
