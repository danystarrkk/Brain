# ¿Qué es SASS?
sass son las iniciales de Syntatically Awesome StyleSheet, es considerado un standard en la industria y es compatible con frameworks css,
## Ventajas
- Orden y Estructura
- Contiene características que no tiene CSS
- Soporta Código CSS
- Menos Código ya que utiliza la “Anidación”
## Desventajas de SASS
- Una nueva sintaxis por aprender
- Se debe copiar con una herramienta especial, no es nativo en los navegadores
- La anidación puede causar problemas si es mucha anidación
