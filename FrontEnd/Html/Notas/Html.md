# Que es HTML?

HTML son la iniciales de “Hypertext Markup Language” Lenguaje de Marcas de Hipertexto, Es un lenguaje, mas no un lenguaje de programación.
HTML5 fue un gran cambio pero ya hoy en día basta con llamarlo html.
Este es un lenguaje fácil de aprender si se pone en practica, es muy intuitivo donde a través de de etiquetas escribiremos código, algunas son:

```html
<p> -> Parrafos
<nav> -> Navegaciones
<header> -> Encabezados de Sitios o elementos
<main> -> Contenido Principal
<footer> -> Pie de página o Elemento
```

La estructura básica de html será la siguiente:

```html
<html>
	<head>
		<title> Mi página Web </title>
	</head>
	<body>
		<p> Contenido de la página </p>
	</body>
</html>
```

De forma básica:

- **head:** colocamos información meta como puede ser información básica del sitio web tal como lenguaje, titulo, descripción y enlaces a hojas de estilo CSS.
- **body:** colocamos todo lo que se verá en la página web, como imágenes, textos, videos y animaciones.

En HTML podemos tener malas practicas y que el lenguaje funciones correctamente, eso no quiere decir que vamos a usar malas practicas ya que google se encarga de penalizar los sitios los cuales se encuentran con malas practicas.
