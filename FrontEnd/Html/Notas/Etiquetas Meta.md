# Etiquetas Meta
Las etiquetas meta nos son una especie de configuración inicial la cual interpreta el navegador para poder leer correctamente el resto del html, algunas de estas etiquetas son:

```html
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1">
<meta name="description" content="Página web de un Blog de Café">
```
