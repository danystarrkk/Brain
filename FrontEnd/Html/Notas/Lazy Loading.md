# Lazy
Usar Lazy load puede ayudar mucho al momento de realizar la descarga de las imágenes o cualquier multimedia en una web, esta es una propiedad que nos permite descargar las imágenes de una web con forme esta las va necesitando, algo necesario es definir un width y height para que funcione y lo podemos definir como:

```html
<img loading="lazy" width="300" height="300">
```
