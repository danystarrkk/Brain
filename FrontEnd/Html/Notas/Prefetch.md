# Prefetch
El Prefetch nos permite cargar la siguiente página que creamos que el usuario visitara o entrara y para esto lo hacemos de la siguiente manera:

```html
  <link rel="prefetch" href="nosotros.html">

```

Pero como sabemos que página precargar? Lo que podemos hacer es mediante google-analítico vamos a ver el porcentaje de cargas en las páginas y a partir de allí vamos a escoger la pagina a cargar de forma inmediata.
